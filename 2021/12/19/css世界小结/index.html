    css世界小结

    发表于 2021-12-19 更新于 2022-01-03 分类于 css
    ##### 文字适应

    文字少居中，多的时候居左

    宽度分离

    采用两层dom来处理定宽样式，这样的好处就是，更改padding 不用更新width。

    个人建议：对于层级较多的dom可以采用，层级过少的就不要增加dom元素了。或者更改为IE盒子

    img标签

    img 标签如果没有 src属性，就是内联元素，等同于span

    可用img标签来处理不加载图片 demo展示

    Css 函数 attr() 可以获取html值

    替换元素

    h1标签可以写上文字，然后用content去显示图片。利于SEO,可屏幕阅读

    空选择器

    :empty 匹配内容为空的元素

    对于内联元素，padding视觉上只有左右，实际情况都会有。但是垂直方向不影响阅读。可以用此扩大点击区域
