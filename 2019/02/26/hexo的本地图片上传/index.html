hexo的本地图片上传

发表于 2019-02-26 更新于 2021-12-19 分类于 工具

原本只是想上传本地图片，结果url一直不对，不过好在还是解决了。一张本地图片引发的血案！！

一· 错误###

最开始，很多人说，直接在_posts中创建images文件夹，把图片放到里面，然后直接引用就好了。
但是根本不行
我认为原因应该是hexo部署到服务器上时，并没有上传这个文件夹。
个人认为应该是hexo的版本问题，3.X版本这个方法是行不通的
没办法，只能继续想办法了

二· 正确###

  1. 插件下载
    首先我们需要hexo的一个插件hexo-asset-image

    当然这之间也会出现一下其他插件丢失的情况，照着报错修改就好了

  2. 修改主题配置文件_config_yml

    post_asset_folder 选项设为 true 来打开。

  3. 这个时候，你再执行hexo new [title]时,会生成一个和md文件名相同的文件夹，如图：

  4. 将你所需要的图片放到这个同名的文件夹里,然后你再使用md语法的图片插入
    例如这里的文件夹名为test,图片名为test.png

  5. 部署到本地服务器
    你会发现图片还是没有。但是你会发现，图片再服务器里面是有的

    那么问题应该还是路径问题了,你会发现，路径是这样的
