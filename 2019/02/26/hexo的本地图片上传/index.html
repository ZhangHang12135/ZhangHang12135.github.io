hexo的本地图片上传

发表于 2019-02-26 | 更新于 2019-09-25 | 分类于 工具

原本只是想上传本地图片，结果url一直不对，不过好在还是解决了。一张本地图片引发的血案！！

一· 错误

最开始，很多人说，直接在_posts中创建images文件夹，把图片放到里面，然后直接引用就好了。
但是根本不行
我认为原因应该是hexo部署到服务器上时，并没有上传这个文件夹。
个人认为应该是hexo的版本问题，3.X版本这个方法是行不通的
没办法，只能继续想办法了

二· 正确

  1. 插件下载
    首先我们需要hexo的一个插件hexo-asset-image

当然这之间也会出现一下其他插件丢失的情况，照着报错修改就好了

  1. 修改主题配置文件_config_yml

    post_asset_folder 选项设为 true 来打开。
