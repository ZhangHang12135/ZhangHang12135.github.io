发表于 2019-03-05 | 更新于 2019-09-25 | 分类于 Bootstrap

优化结构

Boostrap优化了表格结构标签，仅仅优化以下标签:

<table>: 定义表格容器，构建表格数据的框架。
<thead>: 定义表头容器
<tbody>: 定义表格内容容器
<tr>: 定义数据行结构，即表格的每一行
<td>：定义表格数据的单元，即每一格
<th>：定义每列（或行）所对应的标签（label），即定义列标题或行标题
<caption>：定义表格标题，用于对表格进行描述或总结，对屏幕阅读器特别有用

由于table标签广泛使用，Bootstrap3.0规定，只有添加.table类样式才行

注,这里插入一下nth-child和nth-of-type区别

:nth-of-type(n) 选择器匹配属于父元素的特定类型的第 N 个子元素的每个元素.n 可以是数字、关键词或公式。
:nth-child(n) 选择器匹配属于其父元素的第 N 个子元素，不论元素的类型。n 可以是数字、关键词或公式。

注,这里插入一下各选择器的用法

h1 em {background-color:red} 后代选择器：这里的表示整个文档中的h1后面的em元素为红色背景
h1>em {background-color:red} 子元素选择器: 这里的表示整个文档中的h1的子元素em元素为红色背景,即第一代儿子，不管什么孙子之后的了
h1+em {background-color:red} 相邻兄弟选择器：这里的表示整个文档中的和h1相邻的em元素为红色背景
h1~em 和+号一样，也是相邻兄弟选择器
